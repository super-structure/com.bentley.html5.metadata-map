Unsupported File Types

Only standard DITA topic files (extension .dita or .xml) are supported.

LwDITA formats and Markdown are not supported (extension .md or .html as well as .xml with the @format='xdita).

Note: Examples of these file types are included here for testing purposes. Warnings should be generated for these in the output log.
